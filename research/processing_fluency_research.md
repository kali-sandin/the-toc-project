# Fluidez perceptiva: cuando leer fácil parece más verdadero

## Hallazgo

La fluidez perceptiva describe la facilidad con la que una persona procesa una señal: texto legible, jerarquía clara, contraste estable y agrupación predecible suelen sentirse más confiables y menos costosos. Cuando esa fluidez baja, el usuario no solo tarda más: también reevalúa si entendió bien, si el elemento pertenece al grupo correcto o si debe volver a mirar.

Para The TOC Project esto es útil porque permite crear incomodidad sin ocultar información: no se trata de hacer el texto ilegible, sino de alternar zonas muy claras con pequeñas desfluideces visibles.

## Fuentes consultadas

- NNGroup, **Legibility, Readability, and Comprehension**: el texto web necesita claridad visual, palabras simples e información fácil de entender; se puede medir y probar.
- NNGroup, **The Aesthetic-Usability Effect**: una interfaz visualmente agradable puede hacer que la gente tolere problemas menores de usabilidad.
- WCAG 2.2, **Visual Presentation / Contrast / Text Spacing** como límites de seguridad: la fricción no debe convertir lectura, foco o espaciado en castigo operativo.

## Cómo empeorarlo con criterio

1. **Ancla fluida**: cada bloque caótico necesita una frase o etiqueta que se lea de una pasada.
2. **Desfluidez localizada**: aplicar microdesajustes a un borde, chip o subtítulo, no a todo el párrafo.
3. **Contraste intacto**: la molestia debe venir de composición y ritmo, no de texto invisible.
4. **Calma limpia**: en modo calma la misma pieza debe recuperar alineación, fondo estable y jerarquía lineal.
5. **Auditabilidad**: si alguien no puede explicar qué aprendizaje aporta la rareza, es ruido y se debe retirar.

## Aplicación actual

Añadir una tarjeta visible “13. Leer fácil parece más cierto” en `#evidence`, enlazada desde el mapa de research. La tarjeta debe poner en tensión una idea: la web parece menos confiable cuando baja la fluidez, pero sigue ofreciendo una ruta de lectura verificable.

## Subtarea recomendada

Auditar después en móvil y modo calma que la nueva tarjeta no introduce overflow, que el enlace al documento se entiende por sus primeras palabras y que la desfluidez visual no baja el contraste ni convierte el texto en adorno.
